# ADR: UI/UX-архитектура мобильного приложения FOOTBAL

## Контекст
Требуется масштабируемая, современная архитектура UI/UX для мобильного футбольного приложения с поддержкой overlay-инфраструктуры, SectionList, фильтров, избранного, истории, кэширования, локализации и тестов.

## Решение
- Использовать кастомную overlay-инфраструктуру (BottomSheet, Toast, GlobalModal).
- SectionList для группировки данных.
- Redux Toolkit + redux-persist для фильтров, истории, избранного.
- MMKV для быстрого кэширования.
- Навигация через StackNavigator, поддержка свайпов.
- Локализация через lingui.js, поддержка dark mode.
- Все компоненты и экраны типизировать, покрыть unit/e2e тестами.

## Альтернативы
- Готовые библиотеки overlay (react-native-bottom-sheet и др.) — ограниченная кастомизация.
- Минималистичный подход — не покрывает требования по overlay, фильтрам, избранному, истории.

## Аргументация
Выбран кастомный стек — максимальная гибкость, контроль над UX, поддержка всех edge cases, масштабируемость.

## Влияние
- Требует больше времени на разработку, но обеспечивает высокое качество UX и готовность к росту продукта.
- Легко расширять, покрывать тестами, интегрировать новые сценарии.
