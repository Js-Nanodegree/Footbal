## Правило хранения устаревших матчей

- При устаревании матчей по TTL не удалять их из кэша, а переводить в статус 'FINISHED' (или 'ARCHIVED').
- Такие матчи не участвуют в live/актуальных секциях, но доступны для истории и аналитики.
- При появлении новых данных по тем же матчам — обновлять статус и поля.
- Это позволяет ещё сильнее сократить количество запросов и хранить историю всех матчей, которые когда-либо были показаны пользователю.

# Единая задача: Сервис обновления, хранения и отображения данных для HomeScreen

## Описание

Реализовать сервис, который обеспечивает:
- Получение данных (матчи, команды, лиги, стадионы) с API или из моков.

- Минимизацию количества запросов к API (не чаще 1 раза в сутки, live-матчи — чаще).
- Корректное отображение live, scheduled и finished матчей на HomeScreen.
- Агрегацию истории матчей без удаления устаревших данных (перевод в статус FINISHED/ARCHIVED).
