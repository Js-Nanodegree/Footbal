## useDisableAnimationsForAndroid

**Назначение:**
Хук возвращает true, если текущая платформа Android (для отключения анимаций), иначе false. Используется для условного отключения анимаций на Android во всех компонентах/хуках, где есть анимации.

**API:**
- `const isDisabled = useDisableAnimationsForAndroid();`

**Контекст:**
- Использовать в анимационных компонентах, чтобы не запускать анимации на Android.
