# StatsBar: context.md

**Назначение:**
Визуальное сравнение статистики двух команд (числа, проценты) с помощью горизонтальных прогресс-баров.

**Варианты:**
- Числа (number)
- Проценты (percent)
- Разные цвета (left, right)

**Цвета:**
Только через statsBarColorMap и tokens/colors.ts.

**Best practices:**
- Inline-цвета запрещены.
- Для текста использовать только Typography.
- Showcase всех вариантов — только в Style Guide.
- Для новых вариантов — расширять statsBarColorMap.
