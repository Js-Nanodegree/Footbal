# TabBar: context.md

**Назначение:**
Нижняя навигация приложения. Поддержка иконок, badge, активных/неактивных вкладок.

**Варианты:**
- Обычный (default)
- С иконками (withIcon)
- С badge (withBadge)

**Цвета:**
Только через tabBarColorMap и tokens/colors.ts.

**Best practices:**
- Inline-цвета запрещены.
- Для текста использовать только Typography.
- Showcase всех вариантов — только в Style Guide.
- Для новых вариантов — расширять tabBarColorMap.
