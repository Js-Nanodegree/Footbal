# Input: context.md

**Назначение:**  
Текстовое поле для ввода данных (поиск, формы, фильтры).

**Варианты:**  
- Обычный  
- С иконкой  
- С ошибкой  
- Disabled

**Цвета:**  
Использует inputColorMap и tokens/colors.ts для всех состояний.

**Рекомендации:**  
- Inline-цвета запрещены.
- Для новых вариантов — расширять inputColorMap.
