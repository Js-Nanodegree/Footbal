# MatchInfoCard: context.md

**Назначение:**
Карточка превью матча: команды, время, дата, стадион. Используется в списках и на экране матча.

**Варианты:**
- С датой
- Без даты
- С названием стадиона

**Цвета:**
Только через matchInfoCardColorMap и tokens/colors.ts.

**Best practices:**
- Inline-цвета запрещены.
- Для текста использовать только Typography.
- Showcase всех вариантов — только в Style Guide.
- Для новых вариантов — расширять matchInfoCardColorMap.
