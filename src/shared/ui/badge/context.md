# Badge: context.md

**Назначение:**
Бейдж/лейбл для статусов, событий, меток. Поддержка разных цветов, размеров, иконок, outline/solid, disabled.

**Варианты:**
- Цвета: primary, secondary, success, error, warning, info, disabled
- Режим: solid, outline
- Размер: small, medium, large
- С иконкой
- Disabled

**Цвета:**
Использует badgeColorMap и tokens/colors.ts для всех вариантов.

**Рекомендации:**
- Inline-цвета запрещены.
- Для новых вариантов — расширять badgeColorMap.
- Showcase всех вариантов — только в Style Guide.
