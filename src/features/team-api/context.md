# Контекст: team-api

**Назначение:**
- Работа с футбольными API (матчи, команды, лиги, standings) через football-data.org v4.
- Строгая типизация (TypeScript), валидация через Zod.
- Enrichment: сбор и нормализация команд, лиг, матчей из API.
- Моки для оффлайн-режима и fallback.

**Актуально на июль 2024:**
- Типы (Match, Team, Competition, Standing, Area, Score, Odds, Referee) полностью синхронизированы с football-data.org v4.
- crestUrl → crest, emblemUrl → emblem, все поля приведены к актуальным названиям и nullable-типам.
- Zod-схемы отражают реальную структуру API.
- Enrichment-логика и сервисы обновлены, работают только с актуальными полями.
- Тесты покрывают все ключевые сценарии, проверяют новые поля (crest, emblem, code, type, shortName, tla, referees, score).
- Устаревшие поля и дубли удалены.

**Архитектурные правила:**
- Строгая типизация, никаких any.
- Все изменения фиксируются в CHANGELOG.md.
- Любые новые поля/эндпоинты — только через обновление типов, схем, enrichment и тестов.
