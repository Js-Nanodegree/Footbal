# match-swiper/context.md

Purpose: Свайпер матчей (MatchSwiper) для главного экрана. Реализует горизонтальный свайп карточек матчей с анимацией, skeleton, обработкой ошибок и пустого состояния. Архитектура FSD: Wrapper (layout), Connector (бизнес-логика, RTK Query), Card (UI), types, hooks, utils. Все данные берутся из RTK Query и AppContext. UI не содержит бизнес-логики. Масштабируемо для добавления новых источников данных и кастомизации карточек.
