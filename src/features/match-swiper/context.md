# Контекст: MatchSwiper

**Назначение:**
Горизонтальный свайпер карточек матчей с анимацией переходов, эффектом стека (как в dating-приложениях), поддержкой до 10 карточек одновременно. Использует MatchCard для отображения информации о матче. При нажатии на карточку — показ алерта со статистикой. Анимация scale/offset для визуализации следующих карточек. Данные берутся из reference (useMatches). Полностью соответствует дизайн-системе.

**Структура:**
- MatchSwiper.tsx — основной компонент свайпера
- hooks/useMatchSwiperAnimation.ts — хук для анимации переходов
- __tests__/ — тесты на свайпы, отображение, обработку нажатий

**UI:**
- FlatList horizontal, pagingEnabled
- Карточка на весь экран, сзади видны края следующих (scale/offset)
- Максимум 10 карточек
- Анимация переходов и визуализация стека
- Использует Typography, фирменные цвета, MatchCard

**Назначение:**
- Быстрый просмотр матчей свайпом
- Визуализация, что есть ещё карточки (эффект стека)
- Переход к статистике (alert)
