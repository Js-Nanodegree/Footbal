# filters/context.md

**Назначение:**

Папка предназначена для хранения чистых функций фильтрации команд и матчей, специфичных для home-screen. Все фильтры реализуются как отдельные функции с типами.

**Использование:**
- Импортировать фильтры в хуки и компоненты home-screen.
- Не содержать бизнес-логики, только фильтрация по параметрам (лига, команда и т.д.).
- Легко покрывать тестами.

**Пример:**
- filterTeamsByLeague — фильтрация команд по id лиги.
- filterMatchesByLeague — фильтрация матчей по id лиги.
- filterMatchesByTeams — фильтрация матчей по выбранным командам.
