# context.md

## Назначение
Локальный AppContext для главного экрана (HomeScreen) хранит состояние фильтров, выбранной лиги, поиска, лоадеров и обеспечивает доступ к этим данным для всех секций и компонентов экрана.

## Структура
- AppContextProvider: провайдер контекста, оборачивает HomeScreen.
- useAppContext: хук для доступа к состоянию.
- Состояния:
  - selectedLeagueId: выбранная лига (persist через MMKV)
  - search: строка поиска
  - loading: состояния загрузки (матчи, команды, лиги)
  - фильтры, сортировки, ошибки

## Best practices
- Использовать только внутри features/home-screen
- Для глобальных данных использовать shared/memory-bank/
- Документировать новые состояния и методы
