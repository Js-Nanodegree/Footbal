**DateSeasonContext:**
Контекст выбора даты/сезона для всех секций экрана. Предоставляет текущую дату/сезон и методы изменения. Используется через хук useDateSeason. Все секции должны получать данные из этого контекста.

[ПРАВИЛО] Для DateSeasonContext тесты писать не обязательно, по решению команды.
